# Travel Explorer

A beautiful, interactive travel exploration website featuring real-time weather integration, location discovery, and inspiring travel photography.

![Travel Explorer screenshot](screenshot.png)

## Features

- **Interactive Map Integration**: Explore destinations and discover travel locations
- **Real-Time Weather**: Check current weather conditions for any city using OpenWeather API
- **Hero Video Background**: Stunning animated hero section with optimized video playback
- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile
- **Smooth Animations**: Elegant fade-ins and smooth scrolling for enhanced user experience
- **Accessibility**: Respects `prefers-reduced-motion` for motion-sensitive users
- **Modern UI**: Clean, minimalist design with gold accents and refined typography

## Setup

1. Clone the repository
2. Replace `YOUR_API_KEY_HERE` in `index.html` with your OpenWeather API key (get one free at [openweathermap.org](https://openweathermap.org/api))
3. Open `index.html` in a web browser

## Usage

- Navigate using the top menu
- Search for any city to see current weather conditions
- Scroll through destination cards to explore travel inspirations
- Click destinations to learn more

## Technologies

- HTML5
- CSS3 (with CSS variables and animations)
- Vanilla JavaScript
- OpenWeather API
- Google Fonts (Playfair Display, Inter)
